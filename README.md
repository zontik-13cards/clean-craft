# ⛏️ Чистый Крафт (Clean Craft)

**Мультиплеерная 2D-песочница** на чистом Flask + ванильный JavaScript.  
Работает **без интернета** — только локальная сеть (Wi-Fi).  

---

## 🎮 Особенности

- **Полноценный мультиплеер** — до 20+ игроков одновременно
- **Система ролей**:
  - 👑 Владелец (owner)
  - ⚡ Админ (admin) — безлимитная дистанция постановки/ломания блоков
  - 🛡️ Модератор (mod) — безлимитная дистанция
  - 👤 Обычные игроки (player)
- **Валютная система** — монетки начисляются за время на сервере
- **Магазин скинов** — покупай скины за монетки
- **Креативный режим** — кнопка `...` в хотбаре, выдаёт 64 блока любого типа
- **Лидерборд** — показывает всех игроков с ролями
- **Система домов** — `/sethome` и `/home`
- **Телепортация** — `/tp <ник>` (для админов/модов)
- **Сохранение** — SQLite, всё сохраняется между перезапусками
